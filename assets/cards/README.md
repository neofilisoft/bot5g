# assets/cards/

โปรเจกต์นี้ไม่แนบอาร์ตการ์ดของบุคคลที่สาม/ทางการมาให้

ถ้าคุณมีสิทธิ์ใช้ภาพการ์ด (เช่น ได้รับอนุญาตจากเจ้าของลิขสิทธิ์โดยตรง หรือเป็นภาพ
ที่คุณวาด/คอมมิชชันเอง) สามารถนำไฟล์ภาพมาวางในโฟลเดอร์นี้ได้เลย โดยตั้งชื่อไฟล์ให้
ตรงกับ card id ในเกม เช่น:

```
assets/cards/5G-001.png
assets/cards/5G-002.png
```

รองรับนามสกุล `.png`, `.jpg`, `.jpeg`

เกมจะตรวจหาไฟล์เหล่านี้อัตโนมัติตอนรัน (`src/ui/cardart.lua`) ถ้าเจอจะใช้ภาพนั้นแทน
การ์ดสไตล์ข้อความที่มีอยู่แล้วทันที ถ้าไม่เจอก็จะ fallback กลับไปใช้แบบเดิมเหมือนปกติ
ไม่ต้องแก้โค้ดส่วนอื่นเลย

หมายเหตุ: card id ทั้งหมดของเกม (5G-001 ถึง 5G-142) เป็นการ์ดต้นฉบับที่สร้างขึ้นใหม่
ไม่ตรงกับ id ของการ์ดทางการ (เช่น BT01-051) ถ้าต้องการใช้ภาพการ์ดทางการจริงๆ
จะต้องสร้างรายการการ์ดใหม่ (ใน `src/cards.lua`) ให้สอดคล้องกับชื่อ/ค่าพลังจริงของ
การ์ดนั้นๆ เอง เพื่อไม่ให้ตัวเลขในเกมขัดกับภาพที่แสดง
